# Strategy Autoresearch Ledger

Last updated: 2026-05-25

This ledger records strategy research as a repeatable idea-result-retrospective loop.
Every accepted idea must use only point-in-time inputs available on the decision date.

## Process

Each iteration writes one Markdown note under `docs/research/iterations/`.

Required sections:

1. Idea
2. Point-in-time contract
3. Buy rule
4. Sell/rebalance rule
5. Result
6. Retrospective
7. Next mutation

## Verification Standard

Run:

```bash
uv run --locked python -m snusmic_pipeline run-sim --warehouse data/warehouse --out data/sim --end 2026-05-22
uv run --locked python -m snusmic_pipeline export-web --warehouse data/warehouse --sim data/sim --out data/web
pnpm --dir apps/web artifact:check
pnpm --dir apps/web typecheck
pnpm --dir apps/web build
```
